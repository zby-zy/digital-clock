# Digital Clock
This project displays the digital clock.

## Compile & Run 
<pre>
         SET TIME
--------------------------
Enter hour: 6
Enter minute: 8
Enter second: 2
</pre>
### Run
<pre>
Clock: 
06:08:02
</pre>
